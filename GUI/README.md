JAVA网络编程

====================================
改造教材19.2和19.3的例子，创建简单的聊天程序
1. 服务器也具有类似客户端的UI界面
2. 服务器也能够向客户端发送信息
3. 客户端和服务器端UI界面要区分各自发送和接收到的信息，比如可以使用颜色区分彼此的聊天记录
